## b2studios' Bubble Party multiplayer engine clinet, modified to added support for Meta (Oculus) VR headsets and it's full native capabilities.

### Controls:
Camera: Headset (Body) Rotation
World movement: Left Joystick
Jump: A (Button 0)
Fire: Right Trigger (Button 15)
Unlink from server: Menu (Button 6)
